Instrucciones para ejecutar:

Backend
-Abrir proyecto backend en vscode.
-Ejecutar xampp y activar MySQL Database y Apache Web Server
-Correr el proyecto con Springboot Dashboard
-Revisar base de datos desde phpmyadmin en el puerto 8080
-Poblar base de datos con el archivo import.sql
-Revisar documentación con Swagger UI. http://localhost:8080/swagger-ui/index.html#/
-Revisar endpoints con POSTMAN. http://localhost:8080/api/productos

Frontend
-Abrir proyecto frontend en vscode (otra ventana).
-Reinstalar npm de ser necesario.
-Comando en terminal de vscode: npm run dev
-Verificar vistas en puerto 5173.
